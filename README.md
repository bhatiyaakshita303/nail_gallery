<h1>Nailavé by Priya</h1>

<p>
Nailavé by Priya is a premium multi-page luxury nail artistry website built using React.js.
The project is designed to showcase a high-end beauty brand with an editorial aesthetic inspired by luxury beauty companies such as Chanel Beauty, Dior Beauty, Paintbox, Glosslab, and Olive &amp; June.
</p>

<div class="section">
<h2>Project Overview</h2>

<p>
This website was created as a fully responsive front-end experience for a luxury nail artist based in Surat, India.
The design focuses on elegance, sophisticated typography, premium visual hierarchy, and modern user experience principles.
</p>

<p>
The platform allows visitors to explore services, view portfolio work, learn about the artist, request appointments, and contact the studio through a seamless and visually engaging experience.
</p>
</div>
